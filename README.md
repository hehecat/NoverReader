# novel

使用 Vue 实现一款小说阅读器，参考了一些Github上开源的项目。

小说内容通过 Scrapy-Redis爬取，后台及接口使用 TP5实现，这里仅是 Vue 项目，仅供参考。

实现功能：

- 用户
  - 保存阅读进度
  - 每日签到
  - 评论、回复、点赞
  - 关注用户
  - 阅读历史
- 阅读
  - 修改背景
  - 修改字体
  - 目录跳转



## 项目安装运行

```
npm install
npm run server
```
